## create

Create a direct link for a template

### Example Usage

```python
from documenso_sdk import Documenso
import os


with Documenso(
    api_key=os.getenv("DOCUMENSO_API_KEY", ""),
) as documenso:

    res = documenso.templates.direct_link.create(template_id=5094.31)

    # Handle response
    print(res)

```
